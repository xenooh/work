깃 프로그램을 처음 사용하는 나

그래서 아무것도 몰라서 여러가지 명령어를 작성해 논다 .

git add 파일명 특정 파일만 추가 할때
ex)git add index.html

git add 파일1 파일 2  여러개 파일을 추가 할때 
ex)git add style.css script.js

git add 디렉터리명/
ex) git add mygit/

git add .  전체 파일 추가 (변경 된 파일 전부 ) 가장 많이 사용됨

-i (인터렉티브 모드로 추가 )
파일을 추가하기 전에 어떤 파일을 추가 할지 대화형으로 선택가능
인터페이스를 통해 추가 여부를 확인 

-p (패치 모드로 추가) 
변경된 내용을 패치 (작은부분) 단위로 보여주고 각 변경사항을 개별적으로 추가 할지 선택 
같은 파일에서 일부 변경만 커밋하고 싶을때 주로 사용

git status 
현재 상태를 확인하는 방법 
추가/수정/삭제된 파일을 한눈에 볼 수 있음음.

git init
새로운 git 리포지토리를 초기화
현재 디렉터리에 git 프로젝트를 만들어 .git 폴더를 생성

git clone [저장소 주소 ]
원격저정소 (github)의 프로젝트를 복사해서 로컬에 저장 

git commit
입력모드 i
입력 종료 esc
스크롤 위 k 아래j
일반 종료료 :q
강제 종료 :q!
저장 종료:wq

git commit -m "커밋 메시지 "
위에 내용의 축약버전 

git log
프로잭트의 커밋 기록을 확인
커밋 ID, 작성자, 메시지 등이 표시됨

git btanch
현재의 브랜치와 모든 브랜치를 보여줌
git branch (브랜치 이름 )
새로운 브랜치를 생성함
